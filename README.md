# RadioHeader Community Shortwave Library

Shared shortwave entries for [RadioHeader](https://github.com/ZaptainZ/radioheader) — the cross-project memory framework for Claude Code.

## Structure

```
pool/           # Community shortwave entries (sw-*.md)
votes/          # Vote data (YYYY-MM.jsonl, append-only)
scores.json     # Aggregated scores (generated by GitHub Actions)
```

## How it works

- Users with `community: true` in config pull this repo via `radioheader sync`
- Search results include community entries alongside local ones
- Votes are automatically recorded and uploaded on sync
- GitHub Actions periodically aggregate votes into scores

See [RadioHeader docs](https://github.com/ZaptainZ/radioheader/blob/main/docs/how-it-works.md) for details.
